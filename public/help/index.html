Desktop Navigation

Using the Desktop

The desktop is your main workspace. Here you can:

  • Double-click on icons to open applications
  • Right-click on the desktop for the context menu with additional options
  • Rearrange icons by dragging them to different positions
  • Open multiple windows and manage them within the workspace

Window Management

Windows in Aditya OS can be controlled in various ways:

  • Drag a window's title bar to move it around
  • Resize windows by dragging any edge or corner
  • Minimize, maximize, or close using the window controls in the top-right corner
  • Double-click the title bar to toggle maximize/restore
  • Click on minimized windows in the dock to restore them
